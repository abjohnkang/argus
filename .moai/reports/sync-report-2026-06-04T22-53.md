## Summary

SPEC-INFRA-001 delivers the runtime foundation that all later Argus features depend on:
a localhost-bound HTTP API in Docker serving Llama 4 Scout inference via Ollama.

**5 REQs implemented:**
- REQ-INFRA-001: API bound to 127.0.0.1; Docker port mapping uses `127.0.0.1:PORT:PORT`
- REQ-INFRA-002: `run_server.sh` pulls images, downloads model, polls `/health` until ready; idempotent
- REQ-INFRA-003: `/health` returns 503 {"status":"loading"} while model loads, 200 {"status":"ready"} when ready
- REQ-INFRA-004: `MODEL` env var overrides model without any source file changes
- REQ-INFRA-005: `LocalhostOnlyMiddleware` rejects non-localhost Host/Origin headers with 403

- `68652ce` feat(infra): Python FastAPI core + unit tests (api/ + pyproject.toml + .env.example)
